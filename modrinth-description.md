## üöÄ Quick Start

### Installation
1. **Install Fabric Loader** for Minecraft 1.20.1
2. **Download WayaCreates Engine** from Modrinth
3. **Place the mod JAR** in your `mods` folder
4. **Install recommended dependencies**:
   - Sodium (for performance)
   - Iris (for shaders)
   - Fresh Animations (for enhanced character animations)
   - Entity Texture Features (for Fresh Animations support)
5. **Launch Minecraft** and join a world

### First Steps
1. **Run `/wayacreates tutorial start`** to begin the interactive tutorial
2. **Press F1** to open the WayaCreates Engine interface
3. **Use Tab** to switch between different editors
4. **Explore the features** and start creating!

## üéØ Usage Examples

### Creating a Cinematic Video
```
/wayacreates video record                     # Start recording
/wayacreates viewport open                    # Open 3D viewport
/wayacreates animate character steve walk     # Animate character
/wayacreates video stop                       # Stop recording
/wayacreates video editor                     # Open video editor
/wayacreates video export                     # Export final video
```

### Streaming with Background Removal
```
/wayacreates livestream start twitch          # Start Twitch stream
/wayacreates livestream background on         # Enable background removal
/wayacreates livestream overlay "Welcome!"    # Add welcome overlay
/wayacreates livestream meme "bruh"           # Add meme overlay
/wayacreates livestream sound "vine"         # Play vine boom
```

### Animating with Fresh Animations
```
/wayacreates viewport open                    # Open 3D viewport
/wayacreates apply fresh zombie steve        # Apply Fresh Animations to zombie
/wayacreates animate face happy              # Animate face expression
/wayacreates export blender                  # Export to Blender
```

## üîß Configuration

The mod creates a configuration file at `config/wayacreates-engine.json` where you can customize:

- **Video Settings** - Resolution, frame rate, quality
- **Audio Settings** - Sample rate, bit rate, channels
- **Recording Settings** - Which overlays to record
- **Livestream Settings** - Default platform, quality
- **AI Assistant Settings** - Enable/disable AI features
